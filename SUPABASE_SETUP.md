# Подключение онлайн-чата

Без Supabase чат работает только в одном браузере. Для общения между телефоном покупателя и устройством администратора:

1. Создайте бесплатный проект на https://supabase.com.
2. Откройте **SQL Editor**.
3. Скопируйте туда содержимое файла `supabase-schema.sql` и нажмите **Run**.
4. Откройте **Project Settings → API**.
5. Скопируйте:
   - Project URL;
   - anon public key.
6. Откройте `supabase-config.js` и замените:

```js
window.VAPE_SUPABASE = {
  url: 'https://ВАШ-ПРОЕКТ.supabase.co',
  anonKey: 'ВАШ_ANON_KEY'
};
```

7. Загрузите обновлённые файлы в GitHub. Vercel автоматически выполнит новый деплой.

Важно: текущие политики базы предназначены для демонстрационного магазина. Перед коммерческим запуском нужна полноценная серверная авторизация администратора.
